# 🚀 NovaTECH Store

Nowoczesny sklep internetowy o futurystycznym wyglądzie stworzony przy użyciu HTML i CSS.

NovaTECH Store to projekt sklepu technologicznego z ciemnym motywem, nowoczesnymi kartami produktów, efektami glow oraz responsywnym układem strony.

# ✨ Funkcje

- 🌙 Nowoczesny ciemny motyw
- 🛒 Karty produktów z efektami hover
- 🔍 Stylowy pasek wyszukiwania
- 📱 Responsywny layout
- 💜 Neonowe efekty w kolorze fioletowym
- 🎨 Gradientowe przyciski
- 🧩 Minimalistyczne menu nawigacyjne
- ⚡ Płynne animacje CSS
- 🖼 Sekcja prezentacji produktów

---

# 🛠 Technologie

- HTML5
- CSS3
- Flexbox

- Font Awesome
